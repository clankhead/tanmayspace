Secure terminal-based chat app using SRP for authentication, ECDH for key exchange, and AES-GCM for encryption, developed as part of the CY6740 course - Network Security.

Real-time encrypted messaging with user login, online user listing, and peer-to-peer communication.

Built-in rate limiting and simple command interface ensure secure and controlled interaction.
